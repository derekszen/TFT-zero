## Reactivation Rule

If an archived track is resumed, do not silently follow the old file. First
write a short active runbook or update `docs/STRATEGIC_LANE.md`,
`docs/LOOP_SCAFFOLD.md`, and `docs/QUALITY_GATE.md` with the current objective,
artifact path, acceptance criteria, and stop rules.

## What Was Consolidated

The active docs now keep only:

- current objective and agent rules;
- compact architecture;
- strategic-lane simulator shape;
- repeatable autonomous loop scaffold;
- deterministic quality gate;
- minimal training command surfaces.
